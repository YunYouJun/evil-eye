
# Evil Eye

在浏览器端追踪眼部观察内容

## Features

- ⚡️ [Vue 3](https://github.com/vuejs/vue-next) + [Vite 2](https://github.com/vitejs/vite) + [TypeScript](https://www.typescriptlang.org/)

## Usage

### Insall

```bash
yarn
```

### Development

```bash
# http://localhost:3333
yarn dev
```

### Build

```bash
yarn build
```
